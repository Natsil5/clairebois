COMTÉ DE CLAIREBOIS - Structure du Projet
==========================================

Racine du projet :
├── index.html                   → Page d'accueil (utilise css/common.css)
├── README.md                    → Documentation complète
├── STRUCTURE.txt                → Ce fichier
│
├── css/
│   └── common.css              → Styles partagés (index + pages annexes)
│
├── js/
│   └── skill-creator.js        → JavaScript du créateur de compétences
│
└── pages/
    ├── character-creator.html  → Créateur de personnage (fonctionnel) ✓
    ├── skill-creator.html      → Créateur de compétences (fonctionnel) ✓
    └── economy.html            → Économie (placeholder)

UTILISATION :
=============

Accès recommandé :
  → Ouvrir "index.html" puis cliquer sur un outil

Accès direct :
  → Ouvrir directement "pages/character-creator.html"

FICHIERS IMPORTANTS :
=====================

✓ index.html                      - Page d'accueil avec menu
✓ pages/character-creator.html    - Créateur de personnage fonctionnel
✓ css/common.css                  - Styles de base
✓ pages/*.html                    - Pages annexes
✓ README.md                       - Documentation détaillée
